# 下载启动

你可以直接pull到本地，或者直接下载压缩包。

前端项目，只需要，在终端使用

```shell
npm install i
```

然后启动前端项目

```shell
npm run dev
```

后端项目，只需要修改配置文件中的key为你的key.
